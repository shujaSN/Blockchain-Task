# Toy Blockchain

## Tested Python Version

- python 3.9.4

## Installation

```
pip install ecdsa
```

## Run Test

```
python -m unittest -v test_wallet.py
python -m unittest -v test_transaction.py
python -m unittest -v test_block.py
python -m unittest -v test_blockchain.py
```

## Run System

```
python main.py
python main_async.py
```
